# 顺源物流管理系统（离线手动保存版）

本系统当前按你的要求运行在**纯前端离线模式**：  
- 不依赖本地服务端  
- 不自动写入 `data/`  
- 通过“手动保存数据包 / 手动读取数据包”实现跨电脑接续

## 运行方式

直接双击打开 `/Users/aotearoa/Desktop/物流管理/index.html` 即可使用。

## 关键保存流程（必须执行）

### 1）下班前（公司电脑）
1. 录入完成后，点击页面上方 **手动保存数据包**。  
2. 将文件保存到项目文件夹内（建议固定文件名：`顺源物流数据包.json`）。

### 2）到家后（家里电脑）
1. 打开同一套项目文件夹内的 `index.html`。  
2. 点击 **手动读取数据包**，选择刚才保存的 `顺源物流数据包.json`。  
3. 在预览弹窗确认后导入，继续工作。

## 重要说明

- 浏览器本地缓存（localStorage）是按浏览器隔离的，换浏览器/换电脑不会自动带过去。  
- 你现在的跨设备连续工作，依赖“手动保存 + 手动读取”这条链路。  
- 顶部状态条若提示“有未保存更改”，请先点击**手动保存数据包**再关闭页面。
